My projects

I maintain several open-source applications of my own. Some are in a better state than others. Below you will find in random order a list of projects I deem noteworthy. Please browse through my GitHub repositories for an exhaustive list.

  • Gcolor3: Gcolor3 is a color selection dialog written in GTK+ 3. It is much alike Gcolor2, but uses the newer GTK+ version and other modernisations to better integrate into your modern desktop.

  • Phosphorus: Phosporus is a wallpaper browser and setter in the spirit of Nitrogen. It can be considered an unofficial port of Nitrogen to GTK+ 3. Phosphorus is different, however, in that it is plugin-based. This allows it be agnostic as to how the wallpaper is actually applied – each Desktop Environment has its own plugin with this functionality.

  • GTimeUtils: a stopwatch and a timer, bundled together as time utilities. Both are written in GTK+ 3 and integrate nicely into your GNOME Shell environment.

  • Arek Window Manager: a plugin to Mutter in an (ambitious) attempt to create a tiling window manager with eye-candy. Due to several limitations on Mutter’s side, a moving Wayland target and a lack of time on my side, this project is not actively maintained at the moment.

  • Powersave: a combination of scripts, udev rules, configuration files and systemd services to enable (aggressive) power saving on a Linux system. The configuration is tailered to my own laptop, but is easily adaptable to a new system.

University projects

Besides maintaining the above projects myself, on my GitHub account you will also find some repositories that hold projects from university. These include, in order of appearance:

  • Excel++: a freshmen object-oriented programming course project to create a spreadsheet application.

  • Ray Tracer: a computer graphics course project to create a ray tracer.

  • 2048: a software engineering course project to create a clone of the once popular game “2048”.

  • Line Follower: an embedded software course project to make a robot follow a line. An Android app is used to send images from the camera to a laptop via the network, where the laptop detects the line and sends commands to the robot via Bluetooth.

  • DNAinator: a “context project” to visualize and explore DNA sequence graphs.

  • Spoofax Shell: the bachelor end project to create a REPL for the Spoofax Language Workbench. Read the accompanying report here.
